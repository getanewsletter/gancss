Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

.foo {
    padding: quarter(3.2px);    // Returns 1
    margin: halve(7.9px);       // Returns 4
    height: double(36px);       // Returns 72
    width: quadruple(3.125px);  // Returns 12
}
